Class: FastMap ​

class.FastMap

快速的Map

  • api的使用方式与原声Map相同
  • 注意为了提速FastMap 的key值不支持引用数据类型

Since

1.62.0

Example

ts
import { FastMap } from 'super-tools-lib'

const map = new FastMap()

map.set('name', '张三')

map.get('name') // 张三
